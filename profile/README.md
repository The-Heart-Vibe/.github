## How we work

- **Trunk-based development** — short-lived feature branches, fast review cycles
- **Stack-agnostic** — pick the right tool for the problem (Vite, Next.js, Node, Astro, whatever fits)
- **AI-augmented** — Claude Code, Cursor, and similar tools are part of our daily workflow
- **Single source of truth** — see our [playbook](https://github.com/The-Heart-Vibe/playbook) for conventions, architecture, and decision log

1. Read the [playbook](https://github.com/The-Heart-Vibe/playbook) — architecture, conventions, runbooks
